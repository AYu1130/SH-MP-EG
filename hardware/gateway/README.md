# 网关硬件 (Raspberry Pi 4B)

存放网关侧硬件资料。推荐按以下方式归档：

```
hardware/gateway/
├── README.md               # 本文件
├── schematics/             # 扩展板 / 转接板的原理图（若有）
│   └── gateway_hat.sch     # 例：KiCad 工程
├── datasheets/             # 相关芯片 / 外设手册 PDF
│   ├── raspberry-pi-4b.pdf
│   └── usb-ble-dongle.pdf
└── photos/                 # 实物接线照片
    └── overall.jpg
```
