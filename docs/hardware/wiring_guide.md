# 硬件接线指南

## 1. 网关（树莓派 4B）

| 接口              | 用途                                                   |
| ----------------- | ------------------------------------------------------ |
| Gigabit Ethernet  | 与路由器 / 上位机有线连接（主通信路径）                |
| Wi-Fi (2.4/5GHz)  | 与 ESP32-S3 节点组网（可与路由器分开，走独立 AP 模式） |
| Bluetooth (CSR)   | 与 STM32F103+HM-10 节点 BLE 通信                       |
| USB-A             | 备用 BLE 加密狗 / ZigBee 协调器                        |
| GPIO              | 预留报警 LED / 蜂鸣器                                  |
| microSD / USB SSD | 系统盘 & SQLite 缓存                                   |

推荐电源：**5V 3A** 官方 USB-C 适配器，避免欠压导致 BLE 扫描不稳定。

## 2. Wi-Fi 节点（ESP32-S3）

| ESP32-S3 引脚 | 外设          | 说明                                                   |
| ------------- | ------------- | ------------------------------------------------------ |
| GPIO 4        | DHT11 S (DATA)| 温湿度传感器数据线；模块板已内置 10kΩ 上拉，裸芯片需外加 |
| GPIO 5        | 蜂鸣器 S/IN   | 有源蜂鸣器信号输入（高电平鸣响，默认有源自带振荡器）    |
| 3V3 / GND     | 电源          | DHT11 与蜂鸣器均支持 3.3~5V，可直接挂 ESP32-S3 的 3V3  |

说明：
- DHT11 采样间隔硬件限制 ≥ 1s（见 datasheet，测量范围 0~50℃ / 20~90%RH，
  精度 ±2℃ / ±5%RH）；
- 有源蜂鸣器"S"脚接 GPIO5，无需 PWM —— 给高电平即发声，低电平静音；
  如使用低电平触发的模块，在 `include/sensors.h` 把 `BUZZER_ACTIVE_HIGH`
  改为 `0` 即可，无需改主程序；
- 若要切换到 DHT22，只需 `platformio.ini` 的 `build_flags` 加 `-DDHT_TYPE=22`。

供电建议：USB 数据口，或外部 5V LDO -> 3.3V。

## 3. BLE 节点（STM32F103ZET6 / 战舰 V4）

| STM32F103 引脚  | 外设          | 说明                                                       |
| --------------- | ------------- | ---------------------------------------------------------- |
| PA2 (USART2_TX) | HM-10 RXD     | 3.3V 直连（RX 侧建议串 1kΩ 保护）                          |
| PA3 (USART2_RX) | HM-10 TXD     | 3.3V 直连                                                  |
| PB7 / PB6       | GY-302 I²C    | SDA / SCL；与板载 24C02 同总线；BH1750 地址 **0x23**（ADDR 接 GND） |
| PE0             | RGB LED R     | 共阴模块，高电平点亮                                       |
| PE1             | RGB LED G     | 共阴模块                                                   |
| PE2             | RGB LED B     | 共阴模块                                                   |
| 3V3 / GND       | 电源          | HM-10 / GY-302 / RGB 模块共地同源 3.3V（GY-302 以模块为准） |

**关于 GY-302（BH1750）**：固件使用 **PB7=SDA、PB6=SCL** 与板载 **24C02** 共用
I²C；BH1750 从机地址 **0x23**（模块 ADDR 接 GND）。照度低于 ``LIGHT_LUX_DARK_MAX``
（默认约 80 lx，见 ``ldr.h``）时 JSON 字段 ``d``=1。若模块地址为 **0x5C**，
在 ``ldr.h`` 或 ``build_flags`` 中覆盖 ``BH1750_I2C_ADDR``。

**关于 RGB LED**：模块为**共阴驱动**（公共脚接 GND，R/G/B 高电平点亮）。
模块内已带限流电阻，可直接由 3.3V GPIO 驱动。

注意：HM-10 必须配置为 **透传 + 从机**模式（默认即是）。上电后用 AT 命令核对：
```
AT           -> OK
AT+NAME?     -> HMSoft    (或自定义 SHMPEG-BLE-01)
AT+ROLE?     -> 0   (slave)
AT+IMME?     -> 1   (等待被连接)
AT+NOTI?     -> 1   (连接/断开通知)
```

## 4. 整体连接拓扑

```text
           ┌────────────────────────────┐
           │       路由器 / AP          │
           └───────┬────────────┬───────┘
                   │ Wi-Fi      │ Ethernet
           ┌───────┴──────┐  ┌──┴────────────────────┐
           │  ESP32-S3    │  │   树莓派 4B 网关      │
           │  DHT22+Buzzer│  │  (Wi-Fi/BLE/MQTT)     │
           └──────────────┘  └──────────┬────────────┘
                                        │ BLE
                              ┌─────────┴─────────┐
                              │ STM32F103 + HM-10 │
                              │   LDR + LED       │
                              └───────────────────┘
```

## 5. 安全与接线规范

- 所有外设共地；
- 长线（>50cm）串联 100Ω 电阻 + 并联 100nF 陶瓷电容做简单去噪；
- 布线远离 220V 强电，避免电磁干扰 BLE / Wi-Fi；
- 每个节点应留调试串口（TTL UART 0.1" 排针 ×4：3V3 / GND / TX / RX）。
